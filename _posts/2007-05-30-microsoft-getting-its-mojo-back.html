--- layout: post title: Microsoft getting it's mojo back ? date: '2007-05-30T23:09:00.000-03:00' author: Alejandro tags: - software - interfaces - microsoft - servers modified_time: '2007-08-09T01:30:29.559-03:00' blogger_id: tag:blogger.com,1999:blog-3848561.post-3395665787792583488 blogger_orig_url: http://alepanizza.blogspot.com/2007/05/microsoft-getting-its-mojo-back.html --- I'm not a big fan of Vista , ( Gaston does not like it either ) and the most notable effect of the switch to Visual Studio 2005 has been my computer's slowdown. :-(

But for all the bad things, Microsoft seems to be getting it's act together, and there is some very cool product been showing off for the near future ( and I'm not talking about pink Zunes, or black XBoxes ).

I've already wrote about Windows Home Server, and I truly think it is a fantastic product; It will be an incredible success if priced right; as more and more people start to pile up Gigabytes of photo/video memories and music collections. WHS provides media server, centralized backup and storage and remote access with simple interface for the home. user.





There is also a nice SDK and a contest ( in typical Microsoft fashion) Code2fame, for writing plug-ins for the home server





The other interesting product is a new UI based on the multi-touch concept shown off a while ago by Jefferson Han of NYU , check out the original concept it's very cool..




The Microsoft version is called Surfacing Computing the Windows "milan" interface goes on top of vista and was demoed today ; it allows to manipulate real time photos and video, with multi touch zooming and moving, if this find it's way into photographers, composition rooms or even homes it's another potential success.
