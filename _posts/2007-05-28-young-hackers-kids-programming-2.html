I began my interest in Ruby, long before the current rails craze.
Now that it has been getting support from everywhere, I had a variety of reasons for revisiting this nice language.
One of the main attractions of Ruby is how easy is to code, and how concise the syntax is.

My new finding is called Hackety hack .
This started as a manifesto: "The little coder's predicament", now it's a programming enviroment for beginners.
It's only available in Ruby for now, but the author says that there will be versions for another languages ( scheme, python ?? ).

The beauty of this interactive tool is that you don't need previous code experience in Ruby or in any language, and that you can write meaningful code : i.e. no calculators or string parsers, but web blog software and mp3 players, really cool stuff.

I think this could be a really cool way to introduce kids to programming, my current favorite is squeak ( another mouse ! ) ; because I think is more abstract/math oriented toward problem solving .
But as a tool to learn Ruby this is really good. Recommended for 13 and up.

Check the official Ruby site.
